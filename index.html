Quick Setup

1

Install

curl -sSL https://github.com/jpeggdev/claudeforge/blob/main/install-remote.sh | bash
2

Configure

Add MCP servers and set permissions via the web interface at localhost:8080

3

Connect

{
  "mcpServers": {
    "proxy": {
      "type": "http",
      "url": "http://localhost:8080/mcp"
    }
  }
}
