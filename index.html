How It Works

Get up and running in minutes with our simple setup process

1

Install ClaudeForge

Deploy the proxy server using Docker or run directly with Node.js

docker run -p 8080:8080 claudeforge/proxy
2

Configure MCP Servers

Add your MCP servers through the web interface and set permissions

3

Connect Your Agent

Point your AI agent to the ClaudeForge proxy endpoint

{
  "mcpServers": {
    "proxy": {
      "url": "http://localhost:8080"
    }
  }
}
